# SH-101 Device Smoke Checklist

1. Install module via `./scripts/install.sh`.
2. Load module in Move Everything Signal Chain.
3. Verify Note On/Off, hold, and transpose behavior.
4. Sweep cutoff/resonance and confirm stable output (no runaway DC).
5. Validate glide and LFO modulation on pads and external MIDI.
6. Save and reload patch, verify parameter persistence.
